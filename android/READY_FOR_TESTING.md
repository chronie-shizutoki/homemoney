## 测试步骤

### 1. 安装 APK
将 `app-debug.apk` 传输到 Android 设备并安装。

### 2. 启动测试

#### 测试 1：欢迎界面
1. 启动应用
2. 应该看到欢迎界面，显示应用名称和欢迎消息
3. 界面应该显示系统默认语言（中文或英文）
4. 应该有两个按钮："语言设置"和"开始使用"

#### 测试 2：语言切换
1. 点击"语言设置"按钮
2. 应该看到三个语言选项：English、简体中文、繁體中文
3. 当前语言应该有高亮背景和 ✓ 标记
4. 点击其他语言
5. 返回欢迎界面，文本应该立即更新为新语言
6. 完全关闭应用（从最近任务中移除）
7. 重新启动应用
8. 应该保持上次选择的语言

#### 测试 3：主界面 WebView
1. 在欢迎界面点击"开始使用"
2. 应该进入主界面
3. **重要**：主界面应该全屏显示 WebView，**没有任何原生标题栏**
4. WebView 应该加载 `http://192.168.0.197:3010` 的内容
5. 右下角应该有一个浮动的设置按钮（齿轮图标）
6. WebView 中的网页应该可以正常交互

#### 测试 4：设置按钮
1. 在主界面点击右下角的设置按钮
2. 应该进入语言设置界面
3. 可以切换语言
4. 返回主界面，WebView 应该继续显示

#### 测试 5：导航
1. 系统返回键应该正常工作
2. 从主界面返回应该回到欢迎界面
3. 从语言设置返回应该回到上一个界面

## 如果测试失败

### 问题：还是看到标题栏
- 确保安装的是最新的 APK
- 完全卸载旧版本后重新安装
- 检查是否是 WebView 内的网页标题栏（这是正常的）

### 问题：WebView 无法加载
- 确保服务器 `http://192.168.0.197:3010` 正在运行
- 确保手机和服务器在同一网络
- 检查手机的网络连接

### 问题：语言切换无效
- 尝试完全关闭应用后重新打开
- 检查是否选择了正确的语言
